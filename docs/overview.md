# Overview

`winesett.com` is Cody Winesett's portfolio site.

The site is intentionally small: a static page, a few visual assets, and Cloudflare Pages configuration. The goal is to keep the public presence direct and credible while making the positioning clearer around product design leadership, AI-assisted workflows, enterprise UX, and design operations.

## Repository role

- Public portfolio site.
- Lightweight reference for current positioning.
- Home base for selected work and contact links.
- Separate from experimental tooling repos such as wireframing, Figma extraction, or content workflow prototypes.

## Boundaries

- Do not add private client details.
- Do not expose credentials, tokens, or deployment secrets.
- Do not overstate production readiness for experiments.
- Keep documentation concise.
